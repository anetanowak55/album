# Photo Album

Static photo gallery for GitHub Pages.

## Updating the Gallery

1. Add a new album folder inside `/photos`.
2. Name the folder with an album title and date, for example:

   ```text
   Urodziny Mateusza 20.05.2025
   ```

3. Add image files to that folder. Supported formats:

   ```text
   .jpg, .jpeg, .png, .webp
   ```

4. Run the gallery indexer from the project root:

   ```bash
   node gallery-indexer.js
   ```

5. Commit and push the updated files to GitHub:

   ```bash
   git add .
   git commit -m "Update gallery"
   git push
   ```

GitHub Pages will serve the updated `index.html`, `style.css`, `gallery.json`, and `/photos` directory.

## Notes

- `gallery.json` is generated by `gallery-indexer.js`.
- Photo dates and descriptions are stored in `gallery.json`.
- Image paths are relative, for example `photos/Album/photo.jpg`, so the site works correctly when deployed under a GitHub Pages project path.
